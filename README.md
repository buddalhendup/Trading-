# Trading Practice Dashboard

This is a beginner friendly practice dashboard for learning target based entries and exits with simulated money only.

## What it includes

- A local web dashboard
- Paper balance tracking
- Target based simulated entry
- Target based simulated exit
- Manual simulated entry and exit buttons
- A trade log saved on your computer

## MacBook setup

```bash
git clone https://github.com/buddalhendup/Trading-.git
cd Trading-
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python3 app.py
```

Then open this in your browser:

```text
http://127.0.0.1:5000
```

This project is for paper practice only. It does not place real orders.
